# Ecommerce-Churn-Analysis
Performed end-to-end churn analysis on 1,200 global e-commerce customers using Python, scikit-learn, Pandas, and Matplotlib — covering EDA, binary classification, and unsupervised clustering.
Built and evaluated 3 machine learning models (Logistic Regression, Random Forest, Gradient Boosting) achieving AUC = 1.000 with the best model, enabling reliable proactive identification of at-risk customers.
Segmented customer base into 4 behavioural groups via K-Means clustering — identifying a New/Low-Value segment of 430 customers churning at 98%, traced to an onboarding failure.
Delivered findings as a full business report, executive slide deck, and interactive analytics dashboard.
